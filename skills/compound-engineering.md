# Compound Engineering — Automated Learning Loops

Make your AI agent learn and improve automatically. Reviews sessions, extracts learnings, updates memory files, and compounds knowledge over time.

Based on work by [lxgicstudios](https://github.com/lxgicstudios).

## How It Works

1. **Nightly review** — scans the day's sessions for patterns and lessons
2. **Extract learnings** — identifies what worked, what failed, what's new
3. **Update memory** — writes durable facts to knowledge graph
4. **Compound** — each day's learnings build on previous ones

## What Gets Extracted

- New tools or techniques discovered
- Failure patterns and their fixes
- Decisions made and their rationale
- New entities (people, projects, services)
- Process improvements

## The Compound Effect

Day 1: Agent knows nothing.
Day 30: Agent has 30 days of accumulated learnings.
Day 365: Agent has a year of institutional knowledge that no human could remember.

The key insight: write everything down, let decay rules handle relevance, and the agent gets meaningfully better over time.
